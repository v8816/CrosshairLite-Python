# CrosshairLite-Python
CrosshairLite — приложение для настройки прицела


Краткое описание
--------------
CrosshairLite — это лёгкое кроссплатформенное приложение на Python с использованием PySide6, позволяющее настраивать и отображать настраиваемый прицел (crosshair) поверх любых окон. Идеально подходит для геймеров, дизайнеров и всех, кому нужен точный визуальный ориентир на экране.

Основные возможности
------------------
* Настраиваемый прицел: выбор стиля (точка, крест, круг, комбинированный), цвета (HEX), толщины, размеров, прозрачности.
* Сцены и пресеты: создание, сохранение и переключение между различными конфигурациями прицела.
* Редактор объектов: добавление пользовательских графических элементов (прямоугольники, круги, треугольники, многоугольники, линии, кресты, X‑кресты) с настройкой позиции, размера, вращения, масштаба, цвета, прозрачности и режима «вырезания».
* Тёмная тема интерфейса с кастомной стилизацией (QSS).
* Поддержка нескольких языков (русский, английский).
* Глобальный хоткей (F8) для показа/скрытия прицела (на Windows).
* Интеграция с Supabase: публикация и поиск сцен в сообществе.
* Предпросмотр сцены в реальном времени.
* Палитра цветов с недавними цветами и предустановленными оттенками.
* Окно‑оверлей, которое всегда поверх других окон и прозрачно для мыши.
* Сохранение настроек локально (JSON).


Технические детали
---------------
* Язык программирования: Python 3.8+.
* GUI‑фреймворк: PySide6 (Qt6).
* Хранение данных: локальные JSON‑файлы (настройки, сцены), Supabase (онлайн‑каталог сцен).
* Бэкенд: Supabase (REST API, Storage).
* Кастомные виджеты: TitleBar, ColorPaletteDialog, ScenePreview, OverlayWindow.
* Поддержка платформ: Windows, Linux, macOS (с ограничениями).
* Локализация: встроенная система i18n (STRINGS).
* Безопасность: SSL‑контекст (certifi).

Структура проекта (ключевые компоненты)
------------------------------------------
* Settings (dataclass): управление настройками приложения и сцен.
* ObjectPainter: отрисовка пользовательских объектов.
* ColorPaletteDialog: диалог выбора цвета с палитрой.
* OverlayWindow / OverlayManager: окно прицела и менеджер для всех экранов.
* TitleBar: кастомный заголовок окна с кнопками управления.
* TextInputDialog: диалог ввода текста.
* ScenePreview: виджет предпросмотра сцены.
* MainWindow: главное окно приложения с вкладками (Прицел, Сцены, Редактор).
* Supabase‑интеграция: методы для работы с REST API и Storage.

Установка и запуск
---------------
1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Установите зависимости:
   pip install PySide6 certifi
3. Клонируйте репозиторий:
   git clone <URL‑репозитория>
4. Перейдите в папку проекта и запустите:
   python main.py

Использование
-----------
1. Запустите приложение.
2. Настройте прицел на вкладке «Прицел» (стиль, цвет, размеры, позиция).
3. Создайте новую сцену на вкладке «Сцены» и сохраните её.
4. Используйте вкладку «Редактор» для добавления и настройки пользовательских объектов.
5. Нажмите F8 для показа/скрытия прицела в игре или приложении.
6. Опубликуйте свою сцену в сообществе через кнопку «Опубликовать».

Вкладки интерфейса
----------------
* Прицел: базовые настройки прицела, язык, кнопка переключения.
* Сцены: управление сценами, пресетами, скрытие базового прицела, редактор в полноэкранном режиме, публикация в сообщество.
* Редактор: работа с объектами (добавление, дублирование, удаление), настройка свойств объекта, предпросмотр сцены.

Горячие клавиши
-------------
* F8: показать/скрыть прицел (на Windows).
